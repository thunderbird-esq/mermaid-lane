Run comprehensive playlist validation and linting for the IPTV project.

Steps:
1. Change to the iptv directory
2. Run `npm run playlist:validate` to validate playlist structure and integrity
3. Run `npm run playlist:lint` to check M3U format compliance using m3u-linter
4. Report any errors, warnings, or issues found
5. If errors are found, provide suggestions for fixing them
6. Summarize the validation results

This command ensures playlists meet quality standards and are compatible with downstream consumers.
